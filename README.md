# 2pchat
just a simple 2pchat that you could download on 2 pc's that i made you could talk to the same pc or 2 different pc's in the same network at port 5000 (you could change that in the code if you want)



HOW TO USE:

1. download the console on one or 2 pc's.


2. if you want to use one pc then execute it 2 times and make one console be the server and the other be a client.


3. if you want to use 2 pc's to communcitae then firstly they must be on the same wifi, then 1 pc starts as a server the other as client, the client pc enters the ipv4 adress of the server pc



4. Your done!
